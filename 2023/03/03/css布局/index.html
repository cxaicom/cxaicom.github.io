css瀑布式布局

  1. css方式
  2. js方式 offsetWidth VS clientWidth
    offsetWidth=width+padding+滚动条+border
    clientWidth=width+padding

    两栏固定中间自适应

  3. flex布局
    思路：父元素设置flex布局，中间元素设置flex:1
  4. 利用浮动
    思路：左右元素分别设置float:left，float:right，中间元素必须置于二者之后
  5. 绝对定位
    左右设置绝对定位(position:absolute)，分别设置left和right属性
  6. 圣杯布局
    利用position:relative和浮动float:left
    优点：结构简单，无多余dom层
    缺点：中间部分宽度小于左侧布局混乱
  7. 双飞翼布局
    利用float:left和新加dom元素
    优点：支持宽高变化，通用性强
    缺点：dom多余结构层，增加渲染树生成的计算量 https://juejin.cn/post/6844903510933258247

中间固定两端自适应

  1. flex布局
    父盒子设置display:flex,两端设置flex-grow:1
  2. 左右利用calc()函数
    左右必须等宽
    http://t.csdn.cn/Cjr6I

上中下布局，中间自适应

  1. flex布局
    父盒子设置display:flex,中间设置flex:1
  2. calc函数
    中间部分高度为calc(100vh-上下盒子总高度)
  3. 绝对定位
    上下设置绝对定位，中间高度为auto
